# 🎮 五子棋 (Gomoku)

一个纯静态的 Web 版五子棋游戏，支持人机对战和双人对战模式。

## ✨ 功能特性

- **双人对战**：本地两人轮流下棋
- **人机对战**：与 AI 对战（简单难度）
- **15×15 标准棋盘**：符合传统五子棋规则
- **响应式设计**：适配桌面端和移动端
- **胜负判断**：自动检测五子连珠
- **重新开始**：一键重置游戏
- **键盘快捷键**：便捷操作

## 🚀 快速开始

### 本地运行

直接打开 `index.html` 文件即可：

```bash
# 方式 1：直接双击打开
open index.html  # macOS
start index.html  # Windows
xdg-open index.html  # Linux

# 方式 2：使用本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### 在线访问

游戏已部署到 GitHub Pages，访问：
```
https://<username>.github.io/gomoku
```

## 🎯 游戏规则

1. 黑棋先行，双方轮流落子
2. 先连成五子（横、竖、斜任意方向）者获胜
3. 点击棋盘交叉点落子
4. 支持悔棋功能

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| R | 重新开始游戏 |
| U | 悔棋 |
| M | 切换游戏模式 |

## 🤖 AI 算法

AI 采用评分制 + 贪心算法：

- **评分系统**：对每种棋型（连五、活四、冲四、活三等）赋予不同分值
- **贪心策略**：评估每个空位的进攻和防守得分，选择最优位置
- **响应时间**：< 1 秒
